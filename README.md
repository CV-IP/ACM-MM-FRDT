# ACM-MM-FRDT
Code repository for our paper entilted "Feature Reintegration over Differential Treatment: A Top-down and Adaptive Fusion Network for RGB-D Salient Object Detection" accepted at ACM MM 2020 (poster).

# Usage Instructions
Requirements

    Ubuntu 18
    PyTorch 1.3.1
    CUDA 10.1
    Cudnn 7.5.1
    Python 3.7
    Numpy 1.17.3
